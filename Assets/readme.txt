1. Чтобы взаимодействовать с UI нужно зажать LEFT SHIFT.
Когда он отжат управление мышкой передается на управление камерой игрока.

2. Чтобы добавить новую характеристику достаточно в Assets/Game/Configs/StatDefinitions/ 
добавить новый SO типа StatDefinition и добавить его в список в StatDefinitions нежащий в 
Assets/Game/Configs/. Характеристика сразу подтянется и в UI и в сохранения. 
Для того, чтобы исспользовать ее в коде достаточно создать реактивное свойство в PlayerModel
и обращаться к нему при необходимости.

3. Слой инфраструктуры позволяет в дальнейшем использовать любые подходы 
к конфигам, локализациям, системам пользовательского ввода сохранений.

1. Версия Unity 2022.3.62f3

2. Чтобы взаимодействовать с UI нужно зажать LEFT SHIFT.
Когда он отжат управление мышкой передается на управление камерой игрока.

3. Чтобы добавить новую характеристику достаточно в Assets/Game/Configs/StatDefinitions/ 
добавить новый SO типа StatDefinition и добавить его в список в StatDefinitions лежащий в 
Assets/Game/Configs/. Характеристика сразу подтянется и в UI и в сохранения. 
Для того, чтобы использовать ее в коде достаточно создать реактивное свойство в PlayerModel
и обращаться к нему при необходимости.

4. Слой инфраструктуры позволяет в дальнейшем использовать любые подходы 
к конфигам, локализациям, системам пользовательского ввода и системам сохранений.

P.S. 
В проекте не применялись подоходы оптимизации (пулы для эффектов, врагов и тд), 
так как в ТЗ об этом не было сказано.
